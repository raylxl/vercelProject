## 已修复缺陷

### BUG-001 外部编码题目为非必填，但生产环境按必填拦截

- 当前状态：已修复关闭。
- 修复结果：外部编码为空时自动生成 `AUTO-...` 编码；TC-014 已生产回归通过。

### BUG-002 SKU 发货数量允许小数提交

- 当前状态：已修复关闭。
- 修复结果：后端与前端共用校验已要求 SKU 发货数量必须为正整数。
- 本地回归：`1.5` 提交返回 HTTP 400，提示“SKU发货数量：必须为正整数”。

### BUG-003 历史外部编码重复校验大小写差异漏检

- 当前状态：已修复关闭。
- 修复结果：提交前按大小写不敏感方式查询历史外部编码。
- 本地回归：先提交 `FIX-HIST-...`，再提交小写编码返回 HTTP 400，提示与历史批次重复；测试数据已清理。

### BUG-004 备注超长未前置校验，提交触发 500

- 当前状态：已修复关闭。
- 修复结果：备注长度超过 256 个字符时在校验阶段拦截。
- 本地回归：260 字符备注返回 HTTP 400，提示“备注：长度不能超过 256 个字符”，不再触发 Prisma 500。

### BUG-005 当前批次内重复外部编码未按重复检测拦截

- 当前状态：已修复关闭。
- 修复结果：同批次外部编码重复时，若收货信息不一致或 SKU 重复，提交前阻止；同一收货信息下的多 SKU 仍允许作为同单多明细提交。
- 本地回归：疑似重复订单返回 HTTP 400；同单多 SKU 不被重复规则误拦截，继续进入“需选择规则”的正常提交前置校验。

### BUG-006 生产接口在考试模式下可匿名执行写操作，正式上线存在权限风险

- 当前状态：已修复关闭。
- 修复结果：新增正式模式访问守卫。默认考试模式仍开放；当 `UNIVERSAL_IMPORT_REQUIRE_AUTH=true` 或 `UNIVERSAL_IMPORT_EXAM_MODE=false` 时，万能导入页面和 API 需要登录。
- 本地回归：在 3001 端口开启 `UNIVERSAL_IMPORT_REQUIRE_AUTH=true` 后，未登录访问 `/api/universal-import/templates` 返回 HTTP 401，提示“请先登录后再操作万能导入”。

### BUG-007 AI 规则手工微调后未按人工修正结果执行

- 当前状态：已修复关闭。
- 修复结果：规则默认值从“仅补齐空字段”改为“覆盖该字段输出”，并同步更新页面提示，确保人工微调优先于 AI 原始映射。
- 验证方式：`npm run build` 已通过；需生产部署后用原 TC-064 样例补录 UI 回归截图。

### BUG-008 预览表格固定表头和横向滚动表现异常

- 当前状态：已修复但需生产 UI 复测。
- 修复结果：预览表格改为 `width: max-content`，预览容器显式横向滚动，表格最小宽度提升到 1760px。
- 遗留验证：需部署生产后用浏览器重新滚动预览表格，确认固定表头和横向滚动均正常。

### BUG-009 断网解析提示暴露原始 Failed to fetch

- 当前状态：已修复关闭。
- 修复结果：统一识别网络 fetch 异常，展示“网络连接异常，请检查网络连接后重试”的业务提示。
- 验证方式：`npm run build` 已通过；建议生产 UI 录屏补证。

### BUG-010 Word `.docx` 上传后 AI 建议返回 500

- 当前状态：已修复但需真实 Word 文件生产复测。
- 修复结果：AI 建议和试解析接口在 Word/文件解析失败时返回 HTTP 422 和明确业务错误，不再 500。
- 遗留验证：需部署生产后用真实 `.docx` 样例复测；若文件可读取，应继续验证 AI 建议规则可生成。

### BUG-011 1000+ 行预览“显示全部/继续加载”未生效

- 当前状态：已修复关闭。
- 修复结果：“继续加载”和“显示全部”会更新渲染数量、性能快照和页面状态。
- 验证方式：`npm run build` 已通过；建议生产浏览器补录大数据量预览操作证据。
